My projects:

Modbus Adapter Demo

This demo simulates a small-scale industrial setup where a legacy PLC exposes temperature data via Modbus TCP, and a software adapter securely polls and translates that data for modern IT consumption. The PLC generates temperature readings based on a time-of-day function with added noise to mimic realistic sensor behavior, storing these values in Modbus holding registers. The adapter connects to the PLC, reads the registers, converts the data into JSON, and could optionally encrypt or forward it to a dashboard or cloud service. This setup demonstrates how legacy OT systems can be integrated into modern monitoring and analytics pipelines without replacing hardware, providing a safe, controlled environment to experiment with IT/OT convergence, real-time polling, data translation, and secure data handling.

C++ Multi-Layer Perceptron (MLP)

After using neural network libraries in Python that were already built for me, I was curious about how the underlying code worked, so I made my own class in C++. I created a basic Pong environment and generated training data for the model. Then I trained the model on the data and allowed it to play the Pong game.

Binary Search Tree (BST)

This was a class project for Data Structures and Algorithms. It demonstrates concepts of binary trees and includes algorithms to insert values, search for values, find the minimum and maximum values, and traverse the tree in order.
