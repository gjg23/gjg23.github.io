C++ Pong Neural Net

This project is a neural network implemented in C++ to play Pong. It demonstrates basic AI and machine learning concepts. First it generates a file of all the possible inputs it could recive and the optimal output move for each. The model is fitted to the data which is supervised learning, and then the model is able to play by controlling the paddle based on the current game state. The model could be improved by making the ball quicker than the paddle and training the optimal move to be where the pong ball will be rather than following the current location at all times. This would also require the model to know the x position and the directon of the ball to predict where the ball will land. You can find the code and detailed explanation in the GitHub repository linked below.

Training Results

This is a simple plot of the loss each training epoch over 500 epochs at a learning rate of 0.00001. The first epochs are able to learn a lot quickly but as it keeps going it is harder and harder to reduce the loss further.

Learning Rate Comparison

This is a log plot of the loss each training epoch over 500 epochs at a learning rates of 0.0001, 0.00001, and 0.000001. Initially the larger learning rate quickly reached a very low loss but was too coarse to keep going and failed. The smaller learing rate learned steadily but too slow to do anything meaningful over 500 epochs. In the middle an acceptable loss was reached. To fully optimize learning, a reduce learing rate function could be implimented.
