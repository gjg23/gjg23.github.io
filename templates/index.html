C++ Pong Neural Net

After using neural network libraries in python that were already built for me, I was curious about how the underlying code worked, so I made my own class in c++. Webscrapper + Generative language model

Curious about how large datasets are created for LLM's I created a very basic webscrapper to find booktitles. Another script is able to use this dataset to train a GPT2 model on the dataset. After this it can generate a booktitle when prompted.
